Privacy Policy

Last updated: March 10, 2026

1. Overview

ClipCraft is a personal desktop application developed and used exclusively by a single individual developer ("I", "me", "my"). It processes video content locally and uploads clips to social media platforms. This application has exactly one user — the developer — and is not publicly available or distributed. This policy explains what data ClipCraft accesses and how it is handled.

2. Data Collection

ClipCraft does not collect, store, or transmit any personal data to any remote servers. There are no backend servers, databases, or analytics services. All video processing happens locally on the developer's machine. Since ClipCraft is a single-user personal tool, no third-party user data is ever collected or processed.

3. Third-Party Platform Access

ClipCraft connects to third-party platforms through their official APIs using OAuth 2.0 authentication. Only the developer's own accounts are connected:

For TikTok: ClipCraft requests permission to upload and publish video content to the developer's own TikTok account via the Content Posting API. No TikTok user data is read, collected, or stored beyond the OAuth tokens needed to complete uploads.

For YouTube: ClipCraft uses the YouTube Data API v3 to upload videos to the developer's own channel and set metadata (title, description, tags, privacy status).

For Instagram: ClipCraft uses the Meta Graph API to publish Reels to the developer's own Instagram account.

4. OAuth Tokens

Authentication tokens are stored locally in a configuration file on the developer's computer. These tokens are never transmitted to any external server. The developer can revoke access at any time through each platform's app settings.

5. Video Content

Video files are processed entirely on the local machine. Content is transmitted directly from the local computer to the destination platform only when an upload is initiated. No intermediary servers are involved.

6. Analytics and Tracking

ClipCraft does not include any analytics, telemetry, tracking pixels, or usage monitoring of any kind.

7. Data Sharing

No user data is sold, shared, rented, or disclosed to any third parties. The only data transmission is the direct upload of video content from the local machine to the selected platform.

8. Data Retention

Since no data is collected, there is nothing to retain on any server. Local configuration and tokens persist on the developer's machine until manually deleted.

9. Children's Privacy

ClipCraft is a personal tool not available to the public and not directed at children under 13. No information from children is collected.

10. Changes to This Policy

This Privacy Policy may be updated from time to time. Changes will be reflected by updating the date at the top of this page.
